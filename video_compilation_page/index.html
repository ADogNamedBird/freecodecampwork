Skateboarding Basic Tricks

There are 3 basic tricks to learn, that will be the building blocks on which many other tricks are derived. Below, we will cover them with embedded videos as a guide.

Shuvit

This is a basic movement to rotate te board 180 degrees on the x axis.

Ollie

This skill allows you to get the board airborn. This can be used to jump up or down obstacles.

Kickflip

This trick combines the Ollie with a flip. This will teach you how to watch the board and build timing to catch it and stop the board's rotation.
